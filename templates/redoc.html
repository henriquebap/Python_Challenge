Documentação da API

Endpoints Disponíveis:

Criar um novo usuário:

  • Método: POST
  • Path: /users
  • Descrição: Área para criar um novo usuário. Se atente em colocar as entradas e informações corretas.

Ler usuário por ID:

  • Método: GET
  • Path: /users/{user_id}
  • Descrição: Lê um usuário pelo ID.

Atualizar usuário por ID:

  • Método: PUT
  • Path: /users/{user_id}
  • Descrição: Atualiza um usuário pelo ID.

Deletar usuário por ID:

  • Método: DELETE
  • Path: /users/{user_id}
  • Descrição: Deleta um usuário pelo ID.

Criar bicicleta para um usuário:

  • Método: POST
  • Path: /users/{user_id}/bikes/
  • Descrição: Cria uma bicicleta para um usuário específico.

Ler bicicleta por ID:

  • Método: GET
  • Path: /bikes/{bike_id}
  • Descrição: Lê uma bicicleta pelo ID.

Atualizar bicicleta por ID:

  • Método: PUT
  • Path: /bikes/{bike_id}
  • Descrição: Atualiza uma bicicleta pelo ID.

Deletar bicicleta por ID:

  • Método: DELETE
  • Path: /bikes/{bike_id}
  • Descrição: Deleta uma bicicleta pelo ID.

Listar todas as bicicletas de um usuário:

  • Método: GET
  • Path: /users/{user_id}/bikes/
  • Descrição: Lista todas as bicicletas de um usuário.

Listar todas as bicicletas:

  • Método: GET
  • Path: /bikes
  • Descrição: Lista todas as bicicletas.

Prever uma imagem de bicicleta:

  • Método: POST
  • Path: /predict-bike-image
  • Descrição: Criação de uma rota para prever uma imagem de bicicleta.
